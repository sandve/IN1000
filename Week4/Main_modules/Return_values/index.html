Retur-verdier

Denne undervisningsmodulen handler om hvordan en subrutine kan returnere en verdi tilbake til kallstedet. Slike subrutiner som returnerer en verdi kalles funksjoner. På kallstedet (hvor funksjonen kalles) sier man at funksjonskallet evaluerer til returverdien.

Begynn gjerne med å se screencasten om temaet. Du kan også bla gjennom slidene eller se på python-koden gjennomgått under forelesingen. Når du har fått en viss oversikt over temaet er det sterkt anbefalt å forsøke å løse oppgaver om temaet. Om du står fast kan du gjerne gå tilbake igjen til forelesing eller slides.
