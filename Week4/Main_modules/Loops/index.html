Løkker

Denne undervisningsmodulen handler om hvordan man kan få et program til å kjøre de samme kodelinjene et stort antall ganger for å oppnå mer omfattende funksjonalitet.

Begynn gjerne med å se screencasten om temaet. Du kan også bla gjennom slidene eller se på python-koden gjennomgått under forelesingen.

Når du har fått en viss oversikt over temaet er det sterkt anbefalt å forsøke å løse oppgaver om temaet tilpasset denne uken (du kan også se på kun basis- , mengdetrenings- eller utfordrings-oppgaver).
Om du kommer tilbake til denne modulen senere i faget (f.eks. for repetisjon) kan du gjerne se på alle oppgaver i Trix knyttet til dette temaet. Om du står fast kan det være fornuftig å gå tilbake igjen til screencast eller slides.
