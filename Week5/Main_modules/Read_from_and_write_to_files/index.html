Read_from_and_write_to_files

Denne undervisningsmodulen viser hvordan man leser fra og skriver til filer. Måten som presenteres er å iterere gjennom linjene i en fil. Modulen viser også hvordan man videre kan strukturere informasjon innad på linjer og på den måten behandle mer komplekse data.

Begynn gjerne med å se screencasten om temaet. Du kan også bla gjennom slidene eller se på python-koden gjennomgått under forelesingen.

Når du har fått en viss oversikt over temaet er det sterkt anbefalt å forsøke å løse oppgaver om temaet tilpasset denne uken (du kan også se på kun basis- , mengdetrenings- eller utfordrings-oppgaver).
Om du kommer tilbake til denne modulen senere i faget (f.eks. for repetisjon) kan du gjerne se på alle oppgaver i Trix knyttet til dette temaet. Om du står fast kan det være fornuftig å gå tilbake igjen til forelesing eller slides.
