Et første Python-program

Denne undervisningsmodulen handler om å komme i gang med å skrive et første python-program. Dette inkluderer å i en editor skrive kode som gjør noe helt enkelt, lagre denne koden som en python-fil og deretter kjøre filen med Python-tolkeren.

Begynn gjerne med å se forelesings-screencast om temaet. Du kan også bla gjennom slidene eller se på python-koden gjennomgått under forelesingen.

Når du har fått en viss oversikt over temaet er det sterkt anbefalt å forsøke å løse oppgaver om temaet tilpasset denne uken (du kan også se på kun basis- , mengdetrenings- eller utfordrings-oppgaver).
Om du kommer tilbake til denne modulen senere i faget (f.eks. for repetisjon) kan du gjerne se på alle oppgaver i Trix knyttet til dette temaet. Om du står fast kan det være fornuftig å gå tilbake igjen til forelesing eller slides.
