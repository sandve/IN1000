Input fra tastaturet

Denne undervisningsmodulen handler om hvordan man i et program kan få informasjon fra brukeren mens programmet kjører.

Begynn gjerne med å se screencasten om temaet. Du kan også bla gjennom slidene eller se på python-koden gjennomgått under forelesingen. Når du har fått en viss oversikt over temaet er det sterkt anbefalt å forsøke å løse oppgaver om temaet tilpasset denne uken. Om du kommer tilbake til denne modulen senere i faget (f.eks. for repetisjon) kan du gjerne se på alle oppgaver i Trix knyttet til dette temaet. Om du står fast kan du gjerne gå tilbake igjen til forelesing eller slides.
